# To-Buy List Manager

## Project Overview
To-Buy List Manager is a simple Python application designed to help users organize their shopping needs. It allows multiple users to create accounts and manage their own private shopping lists. The application saves all data to a text file, ensuring that your list is saved even after you close the program.

## Features
* **User System:** Allows users to Register and Login securely.
* **Private Lists:** Each user has their own separate list.
* **Persistence:** Data is automatically saved to `user_data_pure.txt`.
* **Manage Items:** You can Add, View, and Remove items easily.

## Technologies Used
* Python 3
* File Handling (Text files)
* Command Line Interface (CLI)
